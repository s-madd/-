# Запуск
1. Создайте виртуальное окружение

   - Для Windows:
   ```python -m venv venv```

   - Для MacOS:
   ```python3 -m venv venv```

   - Для Linux:
   ```sudo python -m venv venv```

2. Установите необходимые библиотеки (команды необходимо выполнять в виртуальном окружении)

   - Для Windows:
   ```pip install -r /path/to/requirements.txt```

   - Для MacOS:
   ```pip install -r /path/to/requirements.txt```

   - Для Linux:
   ```pip install -r /path/to/requirements.txt```
